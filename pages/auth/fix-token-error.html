⚠️ Token Authentication Error

Ada masalah dengan token autentikasi Anda

Error Message:
Failed to load resource: the server responded with a status of 403 (Forbidden)
Token tidak valid.

Solusi Cepat:

  1. Klik tombol "Clear Token & Login Ulang" di bawah untuk membersihkan data lama
  2. Anda akan diarahkan ke halaman home
  3. Login kembali dengan email dan password Anda
  4. Token baru akan di-generate dan masalah akan teratasi

Mengapa ini terjadi?

  1. Token Expired: Token JWT memiliki masa berlaku (24 jam). Jika sudah lewat, harus login ulang.
  2. Server Restart: Jika server di-restart dengan JWT_SECRET yang berbeda, token lama tidak valid.
  3. Browser Cache: Token lama masih tersimpan di localStorage browser Anda.
